About the Framework

🧪

Six Consciousness Theories

Tests based on Recurrent Processing, Global Workspace, Higher-Order, Attention Schema, Predictive Processing, and Agency/Embodiment theories from cognitive science research.

🔄

Modular Architecture

Orchestrator system enables parallel test execution and automatic trait discovery. Researchers can extend the framework by adding new theories without modifying existing code.

📊

Dual Evaluation Methods

Combines keyword matching for baseline content coverage with G-Eval (LLM-as-a-Judge) for sophisticated assessment of metacognitive reasoning across different AI models.

⚡

Parallel Processing

Intelligent thread management reduces testing time from 60 to 15 seconds by executing multiple consciousness assessments simultaneously across different models and theories.
